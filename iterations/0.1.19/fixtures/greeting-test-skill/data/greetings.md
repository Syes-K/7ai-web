# 问候语表（Greetings）

供 Agent 通过 `read_skill_file(packId, "data/greetings.md")` 读取。

## 中文 (zh)

- 你好！Skill Pack 0.1.19 测试成功。
- 欢迎回来，目录型技能包已就绪。

## English (en)

- Hello! Skill Pack 0.1.19 test passed.
- Welcome back — your directory skill pack is ready.

## 日语 (ja) — 可选

- こんにちは！Skill Pack のテストに成功しました。

## 使用说明

- 用户说中文 → 从「中文」小节任选一句。
- 用户说英文 → 从「English」小节任选一句。
- 其他语言 → 用英文句 + 说明「表中无该语言，已 fallback」。
