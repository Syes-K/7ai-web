# 用户故事与验收标准：管理后台 i18n（version 0.1.15 愿景 · 批次 3 交付）

本文档为 `prd.md` 子文档，覆盖 `/admin/*` 页面、`AdminShell`、菜单及各子页文案。**建议交付批次：0.1.17**（非 0.1.15 MVP）。

---

## Epic A：管理后台路由与 Shell

### US-A1 管理后台路由迁入 `[locale]`

**作为** 管理员  
**我想要** 通过 `/{locale}/admin/**` 访问管理后台  
**以便** 与站点 locale 策略一致  

**验收标准：**

- [ ] **AC-A1**：`/en/admin/users`、`/zh/admin/config` 等可访问。
- [ ] **AC-A2**：裸 `/admin/**` → 302 至 locale 前缀路径。
- [ ] **AC-A3**：未登录或非管理员 → 按既有策略 redirect（locale 感知）；非白名单 → console 带 `notice=admin_forbidden`。
- [ ] **AC-A4**：`admin` 从 `KNOWN_APP_SEGMENTS` 移除。

### US-A2 AdminShell 壳层双语

**作为** 管理员  
**我想要** 管理后台 Shell 标题、加载态、顶栏链接双语  
**以便** 识别当前环境  

**验收标准：**

- [ ] **AC-A5**：Shell 标题（现网「管理后台」）、加载态、skip link 双语。
- [ ] **AC-A6**：顶栏「对话」「控制台」链接文案与 href 双语 + locale 感知。
- [ ] **AC-A7**：`ConfigProvider locale`、`dayjs.locale` 随 locale 切换。
- [ ] **AC-A8**：顶栏嵌入 `LanguageSwitcher`。

### US-A3 侧栏菜单双语

**作为** 管理员  
**我想要** 管理后台侧栏六项菜单以当前语言展示  
**以便** 导航  

**验收标准：**

- [ ] **AC-A9**：配置管理、用户管理、模型管理、提示词模版、日志查询、系统助手管理 — 全部 i18n key 化。
- [ ] **AC-A10**：菜单 path locale 感知。

## Epic B：用户管理（users）

### US-B1 用户列表与只读账号操作双语

**作为** 管理员  
**我想要** 用户表格、只读开关、重置密码等操作文案双语  
**以便** 管理账号  

**验收标准：**

- [ ] **AC-B1**：列头（邮箱、昵称、角色、只读状态等）双语。
- [ ] **AC-B2**：只读 Tag（「只读」「读写」）、操作按钮（「设为只读」「取消只读」）双语。
- [ ] **AC-B3**：确认弹窗（「设为只读账号？」等）标题、正文、按钮双语。
- [ ] **AC-B4**：成功 toast（「已设为只读账号」等）双语。
- [ ] **AC-B5**：不能变更当前登录账号只读状态的 tooltip 双语。
