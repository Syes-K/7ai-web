## 待产品确认（开放问题）

1. **MCP 挂载与 RAG 是否解耦**：是否采纳 PRD 默认「提案 A」（每轮均挂载助手绑定知识库上的 MCP，不依赖向量检索是否命中），还是「提案 B」（仅在某条件下挂载）。
2. **每用户 / 每知识库 MCP 数量上限**：是否需要硬限制（防工具爆炸与滥用）。
3. **删除 MCP 配置策略**：若仍被知识库引用，是 **禁止删除** 还是 **允许删除并自动解除所有挂载**（后者需审计日志）。
4. **`enabled = false` 的 MCP**：被知识库挂载时，是 **对话忽略并 UI 警告**、还是 **禁止保存知识库**、或 **自动从挂载列表剔除**。
5. **连接测试的触发方式**：是否仅手动；是否需要定时后台探测（涉及任务队列与通知，通常非首版）。
6. **非助手会话**：若未来存在「会话级选知识库」，MCP 是否一并纳入；**本期是否明确声明不支持** 即可。
7. **工具调用可观测性**：对用户是否展示「本轮已加载 MCP 列表 / 某 MCP 失败」的显性提示文案级别（仅日志 vs Toast vs 消息内脚注）。
