# 自测清单：对话页（0.0.6）

在已配置 `CHAT_LLM_API_KEY` 与可用模型前提下：

- [ ] 未登录访问 `/chat` → 重定向登录。
- [ ] 登录后进入 `/chat` → 加载会话列表；有历史时默认选中第一条并显示消息。
- [ ] 点击「新建对话」→ 出现新会话并可选中，主区为空，可发送首条消息。
- [ ] 发送消息 → 用户气泡先出现，助手内容**流式**逐字出现，结束后列表标题/预览随刷新更新。
- [ ] 切换另一条会话 → 消息区切换无串会话；输入框草稿按会话独立。
- [ ] 「清空当前对话」→ 确认后消息为空，列表仍保留该会话，标题回退为「新对话」类展示。
- [ ] 窄屏 → 「历史」打开 Drawer，选中后关闭 Drawer。
- [ ] 越权/无效会话 ID（若手动改 URL）→ 接口 404 时前端提示错误。
