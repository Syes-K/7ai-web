# 阶段 3B 实现说明（version 0.1.10）

## 本轮增量需求（公安备案整改）

### 1) 注册接口改为管理员受控

- `POST /api/auth/register` 增加鉴权：
  - 未登录返回 `401 UNAUTHORIZED`（`请先登录管理员账号`）；
  - 已登录但非管理员返回 `403 FORBIDDEN`（`仅管理员可创建账号`）。
- 管理员判定复用 `src/server/auth/admin.ts` 的 `isAdminEmail`（`ADMIN_USER` 白名单）。

### 2) 安全策略说明

- 前端页面守卫仅作体验收敛；后端接口保留强校验防绕过。
- 注册能力由管理员统一执行，满足备案场景下账号创建可控性要求。
