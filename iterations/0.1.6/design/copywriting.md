# 文案稿：知识库管理 / 向量化状态 / 分片测试 / 控制台助手配置（0.1.6）

> 原则：短句、可执行、避免技术细节泄露；错误提示优先给用户可行动的下一步。  
> 范围提醒：**本期仅支持直接文本输入**；文件上传为禁用占位提示。

## 1) 通用按钮与页面标题

- 列表页标题：知识库
- 按钮：
  - 新建知识库
  - 保存并向量化
  - 保存
  - 取消
  - 编辑
  - 刷新状态
  - 重试向量化
  - 测试分片
  - 开始测试
  - 重置
  - 复制分片内容
  - 清空选择
  - 完成

## 2) 创建 / 编辑 / 保存反馈

### 2.1 成功

- 保存成功（新建/编辑通用）：保存成功
- 保存成功且进入向量化：保存成功，正在向量化…
- 复制成功：已复制

### 2.2 失败（提交级）

- 保存失败（通用）：保存失败，请稍后重试
- 保存失败（网络/服务异常可选更具体）：网络异常，保存未完成，请重试

### 2.3 放弃修改（二次确认）

- 标题：放弃修改？
- 说明：你有未保存的更改，离开后将丢失。
- 按钮：继续编辑 / 放弃离开

## 3) 表单校验文案（字段级）

### 3.1 必填

- 名称必填：请填写名称
- 正文必填：请填写正文内容

### 3.2 长度与格式

- 名称过长：名称最多 64 个字符
- 描述过长：描述最多 500 个字符
- 正文超限：正文已超出上限（{current}/{max} 字符），请删减后再保存

### 3.3 名称唯一性

- 名称冲突：名称已存在，请更换名称

### 3.4 tags 约束

- tags 超过数量：最多添加 20 个标签
- 单个 tag 过长：单个标签最多 20 个字符
- tag 重复（可选）：该标签已添加

### 3.5 内容格式提示（帮助文案）

- 内容格式说明：不同格式会影响分片方式与检索效果
- Markdown 提示（可选）：Markdown 将按标题/结构进行分片

## 4) 向量化状态（详情页/列表页）

### 4.1 pending

- 状态标题：向量化中
- 状态说明（短）：正在处理内容，完成后即可用于检索与分片测试
- 轮询超时提示（可选）：仍在处理中，可稍后再来或手动刷新

### 4.2 success

- 状态标题：已完成
- 状态说明（可选）：已可用于聊天检索与分片测试
- 时间字段标签：最近完成时间

### 4.3 failed

- 状态标题：向量化失败
- 状态说明（通用）：暂不可用于检索与分片测试
- 失败原因字段标签：失败原因
- 重试按钮提示（tooltip/说明）：重试会重新生成分片与向量数据

### 4.4 刷新/重试失败

- 刷新失败：刷新失败，请重试
- 重试失败：重试未成功，请稍后再试

## 5) 文件上传预留入口（禁用态）

### 5.1 标题与说明

- 区块标题：文件上传（即将支持）
- 说明（主）：本期仅支持直接文本输入，文件上传暂不可用
- 说明（补充，可选）：后续将支持上传 PDF/TXT/MD 自动解析为可检索内容

### 5.2 禁用按钮/占位

- 按钮：上传文件（PDF/TXT/MD）
- tooltip（hover 可选）：即将支持文件上传，本期暂不可用

## 6) 分片测试（Chunk Test）

### 6.1 抽屉/弹窗标题与说明

- 标题：分片测试
- 说明：用于验证分片与召回效果（不生成回答）

### 6.2 Query 输入

- 标签：测试问题（Query）
- placeholder：输入一段用于检索的文本，例如“我们的退款规则是什么？”
- 必填提示：请输入测试问题

### 6.3 参数区（topK/阈值）

- topK 标签：topK
- topK 帮助：最多展示 topK 条结果
- 阈值标签：阈值
- 阈值帮助：低于阈值的结果将被过滤

### 6.4 参数非法提示（配置非法提示）

- topK 非法：topK 必须为正整数
- topK 过大（可选）：topK 最大为 {max}
- 阈值非法：阈值需在 0 到 1 之间

### 6.5 loading / 摘要

- loading：正在检索…
- 摘要模板（建议）：命中 {hit} 条（过滤后展示 {shown} 条，topK={topK}，阈值={threshold}）

### 6.6 结果空态

- 标题：未找到匹配分片
- 说明（推荐默认）：你可以尝试换一种问法，或降低阈值后再试
- 次要建议（可选）：确认该知识库已完成向量化

### 6.7 请求失败/错误态

- 错误提示（通用）：检索失败，请稍后重试
- 重试按钮：重试

### 6.8 前置条件提示（pending/failed）

- 向量化中不可测：向量化进行中，完成后可进行分片测试
- 向量化失败不可测：向量化失败，请先重试向量化后再测试

## 7) 控制台：助手管理知识库多选器

### 7.1 摘要态

- 未选择：知识库：未选择
- 已选择（模板）：知识库：已选 {count} 个
- 按钮：选择知识库

### 7.2 选择面板

- 标题：选择知识库
- 搜索 placeholder：搜索名称或描述
- 空列表（无数据）：暂无知识库，请先去创建
- 空搜索结果：没有找到匹配的知识库

### 7.3 不可用项提示（failed）

- 状态文案：不可用
- 说明（tooltip/副文案）：向量化失败，暂不可用于检索
